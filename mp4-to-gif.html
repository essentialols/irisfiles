Yes, as long as your browser can play the MP4 file. MP4 with H.264 video is supported by all modern browsers. MP4 files using H.265 (HEVC) may only work in Safari and recent versions of Chrome. The converter handles any resolution and automatically scales the output.

MP4 uses highly efficient inter-frame compression (H.264), while GIF stores each frame independently with only simple LZW compression. A 2MB MP4 can easily produce a 20MB GIF. To keep the GIF small, reduce the width to 320px, lower FPS to 8, and trim to the shortest clip you need.

The converter extracts only the video track from your MP4 file. Audio is ignored since the GIF format does not support sound. Your original MP4 file is not modified in any way.

There is no strict file size limit since nothing is uploaded. However, your browser needs enough memory to decode the video. For best results, keep MP4 files under 100MB and trim to 60 seconds or less using the start and end time controls.
