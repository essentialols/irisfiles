Video files can contain metadata including video codec (H.264, H.265, VP9, AV1), resolution, frame rate, bitrate, audio codec, sample rate, channel layout, creation date, encoder software, and tags like title, artist, and copyright. This data is embedded in the container format (MP4, MKV, WebM, etc.).

The stripped file should be very close to the original size, typically a few kilobytes smaller due to removed metadata. If the size differs significantly, the container may have been remuxed slightly differently, but the video and audio quality remain identical.
