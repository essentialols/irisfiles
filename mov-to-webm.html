Frequently Asked Questions

WebM is an open, royalty-free format optimized for the web. Converting MOV files to WebM is ideal for embedding videos on websites, uploading to platforms that prefer open formats, or reducing file size for web delivery.

IrisFiles uses FFmpeg.wasm, a WebAssembly build of FFmpeg. It decodes the video and audio tracks from your MOV file and re-encodes them as VP8 video and Vorbis audio in a WebM container. All processing happens locally.

Conversions are limited to 100MB and 10 minutes. MOV files from iPhones can be large, so check your file size first. For bigger files, use a desktop tool like HandBrake or VLC.
