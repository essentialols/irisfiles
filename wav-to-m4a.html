Frequently Asked Questions

Your browser reads the uncompressed WAV audio using the Web Audio API and encodes it as M4A (AAC in an MP4 container). No server is involved, so your files remain completely private.

M4A uses the AAC codec, which is lossy, so there is some quality reduction compared to the lossless WAV source. However, AAC at typical bitrates is perceptually transparent for most listeners, and the file size savings are dramatic.
