Frequently Asked Questions

Your browser decodes the AAC audio using the Web Audio API and re-encodes it as OGG Vorbis. Everything runs locally, so no files are uploaded to any server.

Both AAC and OGG Vorbis are lossy codecs, so transcoding introduces a small generational loss. In practice the difference is minimal, and OGG provides broad compatibility with open-source software and Linux devices.
