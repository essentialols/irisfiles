Frequently Asked Questions

IrisFiles uses FFmpeg.wasm, a WebAssembly build of FFmpeg that runs entirely in your browser. It decodes the MKV file and re-encodes it as MP4 using H.264 video and AAC audio. No server is involved.

There is a small quality reduction because the video is re-encoded. IrisFiles uses a high quality preset (CRF 23 with the fast preset) that preserves most visual detail. For lossless conversion, a desktop tool that can remux MKV to MP4 without re-encoding would be needed.
