About IrisFiles

IrisFiles is a free file converter that works entirely in your browser. Your files are never uploaded to a server. Every conversion happens on your device using WebAssembly and the browser's built-in APIs.

Why this exists

In March 2025, the FBI issued a public warning about free online file converters being used to distribute malware. The warning highlighted a fundamental problem: most online converters require you to upload your files to a remote server. You have no way of knowing what happens to your files after they leave your device.

IrisFiles takes a different approach. There is no server. The entire application is static HTML, CSS, and JavaScript. When you drop a file into IrisFiles, it stays on your device. The conversion runs in your browser, and the result is downloaded directly, just like any other file.

What it can do

IrisFiles currently has 128 tools across several categories:

  • Images: Convert between 10 formats (HEIC, AVIF, PNG, JPG, WebP, GIF, BMP, TIFF, ICO, SVG). Also compress, resize, strip EXIF data, view metadata, and combine images into an animated GIF.
  • Video: Convert between MP4, WebM, MOV, AVI, and MKV, or extract to GIF. Compress video with quality presets and optional resolution scaling.
  • Audio: Convert between MP3, WAV, OGG, FLAC, M4A, and AAC. Compress audio with standard bitrate tiers.
  • PDF: Merge PDFs, split a PDF into pages, convert PDF pages to images, and assemble images into a PDF.
  • Documents: Convert EPUB, RTF, DOCX, and MOBI files to plain text or PDF.
  • Fonts: Convert between TTF, OTF, and WOFF.
  • Archives: Create and extract ZIP files.
  • Unknown files: Drop any unrecognized file to identify its type. IrisFiles checks magic bytes against a 600+ format signature database and shows you what it found.

How it works

Image conversion: For common formats (JPG, PNG, WebP, etc.), IrisFiles loads the image into a Canvas element and re-encodes it using the browser's built-in encoder. For HEIC/HEIF (iPhone photos), it lazy-loads a WebAssembly build of libheif (version 1.21.2) from the CDN. AVIF conversion also uses a WASM decoder when needed.

Video and audio conversion: IrisFiles lazy-loads FFmpeg.wasm (version 0.12, roughly 10MB) from the CDN only when you use a video or audio tool. FFmpeg runs entirely in your browser; no media ever leaves your device. MP3 encoding uses lamejs; WAV encoding uses the Web Audio API directly.

PDF operations: Merge and split use pdf-lib. Image-to-PDF uses jsPDF. PDF-to-image rendering uses pdfjs-dist. All lazy-loaded from the CDN.

Format detection: IrisFiles reads the first few bytes of every file (magic bytes) and checks them against a 601-entry signature database. It never relies on file extensions.

Batch downloads: Multiple output files are packaged into a ZIP using fflate, assembled entirely in your browser.

What about quality?

Image quality sliders are continuous (1-100%) with snap points at common values. The default is 100%, meaning full quality with no extra compression applied. Lower the slider to trade some visual fidelity for a smaller file. Video compression uses CRF-based presets (High, Medium, Low). Audio compression uses standard bitrate tiers. Some tools (image-to-GIF, PDF, document, font, and ZIP tools) have no quality control because their output format uses fixed encoding.

Technology

  • HEIC decoding: heic-to (WASM, libheif 1.21.2), lazy-loaded from CDN
  • Image encoding: Browser Canvas API (toBlob)
  • Video and audio processing: FFmpeg.wasm 0.12 (~10MB WASM), lazy-loaded from CDN
  • GIF encoding: gifenc (local bundle)
  • Audio encoding: lamejs for MP3, Web Audio API for WAV, FFmpeg.wasm for OGG/FLAC/M4A/AAC
  • PDF operations: pdf-lib (merge/split), jsPDF (image-to-PDF), pdfjs-dist (PDF-to-image)
  • EXIF metadata: ExifReader + piexifjs (lossless JPEG EXIF editing)
  • Font parsing: opentype.js
  • ZIP packaging: fflate (~8KB gzipped, local bundle)
  • Format detection: Magic bytes, 601-entry signature database
  • Hosting: Static files on Vercel (zero server-side code)
  • Privacy: No analytics, no tracking, no cookies
