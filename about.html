About ConvertFast

ConvertFast is a free image converter that works entirely in your browser. Your files are never uploaded to a server. Every conversion happens on your device using WebAssembly and the Canvas API.

Why this exists

In March 2025, the FBI issued a public warning about free online file converters being used to distribute malware. The warning highlighted a fundamental problem: most online converters require you to upload your files to a remote server. You have no way of knowing what happens to your files after they leave your device.

ConvertFast takes a different approach. There is no server. The entire application is static HTML, CSS, and JavaScript. When you drop a file into ConvertFast, it stays on your device. The conversion runs in your browser, and the result is downloaded directly, just like any other file.

How it works

For HEIC/HEIF files (iPhone photos): ConvertFast loads a WebAssembly build of libheif (version 1.21.2), the reference HEIC decoder. The WASM binary is lazy-loaded on the main thread only when you actually need it (roughly 2.5MB). It decodes the HEIC file, then re-encodes it as JPG or PNG.

For JPG, PNG, and WebP files: Modern browsers already know how to decode these formats. ConvertFast loads the image into a Canvas element, then exports it in the target format using the browser's built-in encoding. This takes milliseconds, even for large images.

For batch downloads: Multiple files are packaged into a ZIP using fflate, a fast JavaScript compression library. The ZIP is assembled in your browser and downloaded as a single file.

What about quality?

Every converter page includes a quality slider (10-100%). The default is 92% for lossy conversions (JPG, WebP) and 100% for lossless conversions (PNG). At 92%, visual quality is excellent and file sizes stay reasonable. Lower the slider to get smaller files at the cost of some visual fidelity.

Technology

  • HEIC decoding: heic-to (WASM, libheif 1.21.2)
  • Image encoding: Browser Canvas API (toBlob)
  • Format detection: Magic bytes (not file extension)
  • ZIP packaging: fflate (~8KB gzipped)
  • Hosting: Static files on Vercel (no server-side code)

Open source

ConvertFast is open source. You can inspect the code, verify that no files are uploaded, and contribute improvements. The entire application is under 100KB of custom code (excluding the HEIC WASM binary).
